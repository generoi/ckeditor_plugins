CKeditor Plugins
================

- Fontawesome (under development)
- Snippets, allow you to add code snippets thorugh ckeditor.config.js and have
  them appear in a dropdown.

  config.snippets = [
    ['<span></span>', 'insert span', 'insert span']
  ];

Installation
------------
1. Enable
2. Add a snipets configuration to your themes ckeditor.config.js
3. Enable the plugin in the ckeditor settings for your filters, eg.
   admin/config/content/ckeditor/edit/Full.
